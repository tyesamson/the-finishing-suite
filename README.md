# The Finishing Suite

Website built with Eleventy and TailwindCSS.

## Installation

```
yarn install
```

## Development

```
yarn run dev
```

or if you're developing on Windows:

```
yarn run dev:windows
```

## Deployment

```
yarn run build
```

or if you're building on Windows:

```
yarn run build:windows
```

Then copy everything from `dist`.
